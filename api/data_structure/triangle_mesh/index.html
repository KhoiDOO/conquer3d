    TriangleMesh

    conquer3d.data_structure.TriangleMesh

    Bases: pybind11_builtins.pybind11_object

    A highly efficient Triangle Mesh data structure natively backed by CUDA.

    Parameters:

    • in_vertices (Tensor) –

      A tensor of shape (N, 3) containing float32 vertex coordinates.

    • in_triangles (Tensor) –

      A tensor of shape (M, 3) containing int32 triangle indices.

    • in_vertex_normals (Tensor) –

      A tensor of shape (N, 3) containing float32 vertex normals. Defaults to None.

    • in_vertex_colors (Tensor) –

      A tensor of shape (N, 3) containing float32 vertex colors. Defaults to None.

    conquer3d.data_structure.TriangleMesh.__module__ = 'conquer3d._C' class-attribute

    sample_points(self: conquer3d._C.TriangleMesh, num_points: int, uniform: bool = False, return_normals: bool = False, return_colors: bool = False, use_triangle_normal: bool = True) -> tuple[torch.Tensor, torch.Tensor, Optional[torch.Tensor], Optional[torch.Tensor]]
